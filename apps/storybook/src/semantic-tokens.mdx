    # Semantic color tokens

    Stalk UI semantic colors use Panda’s `base` / `_dark` pairs, which compile to CSS variables under **`.dark`** on the document root. Storybook’s **mode** toolbar sets `data-color-mode` and toggles the **`dark`** class on the root element so variables match the toolbar. Values below resolve through the Panda `token()` helper for the active mode.

    Surfaces and canvas fills.

    ## Vivid (saturated indicators)

    A saturated, **mode-stable** step (Radix 9) for status text on dark surfaces and saturated indicators (dots, tags, heat cells). Pair `*.vivid` fills with `*.vividContrast` for legible text — `warning.vividContrast` is dark text, the others are light, and neither flips with color mode.

    <ColorPalette>
      <ColorItem
        title="Vivid"
        subtitle="Saturated, mode-stable fills (step 9)"

          accent: token('colors.accent.vivid'),
